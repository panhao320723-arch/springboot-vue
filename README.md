# 校园失物招领互助平台

这是一个基于 Spring Boot + Vue 3 的毕业设计项目。

## 项目结构

- `backend/`: 后端 Spring Boot 项目
- `frontend/`: 前端 Vue 3 项目

## 快速开始

### 后端

1. 进入 `backend` 目录
2. 确保配置好 MySQL 数据库（修改 `src/main/resources/application.yml` 中的数据库配置）
3. 运行 `CampusLostAndFoundApplication`

### 前端

1. 进入 `frontend` 目录
2. 运行 `npm run dev`
3. 访问 http://localhost:3000

## 功能模块 (计划中)

- 用户注册/登录
- 失物发布
- 招领发布
- 首页信息流
- 搜索与筛选
- 个人中心
